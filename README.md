## 1. Обзор проекта
«Вкус» — это концептуальное мобильное приложение для электронной коммерции, ориентированное на сегмент полезных перекусов и здорового питания. Приложение разработано с использованием современного декларативного подхода к созданию интерфейсов — **Jetpack Compose**, что обеспечивает высокую скорость работы и плавность анимаций.

Цель проекта: создание удобной платформы для выбора товаров, анализа локаций и получения актуальной контекстной информации (погода, геопозиция) в едином интерфейсе.

---

## 2. Функциональные возможности

### 2.1. Каталог продукции (Feed)
* **Главный экран:** Динамическая лента товаров с разделением на категории (популярное, новинки, полезное).
* **Карточки товаров:** Информативные превью с изображением, ценой и кратким описанием.
* **Детализация:** При выборе товара открывается экран с подробными характеристиками, составом и энергетической ценностью.

### 2.2. Поиск и фильтрация (Search)
* Интеллектуальный поиск по названиям продуктов.
* Категоризированный просмотр (например, «Чипсы», «Напитки», «Сладости»).
* История недавних запросов для быстрого доступа к любимым позициям.

### 2.3. Интерактивная карта и Геосервисы (Map)
* **Интеграция OpenStreetMap:** Полноценный картографический модуль для поиска ближайших точек выдачи или партнерских магазинов.
* **Смарт-маркеры:** Установлены метки ключевых объектов с поддержкой информационных окон (название, статус).
* **LBS (Location Based Services):** Отображение текущей геопозиции пользователя в реальном времени.

### 2.4. Информационный модуль (Weather API)
* **Динамический мониторинг:** Автоматическое получение метеоданных для текущего региона через OpenWeather API.
* **Интерфейс:** Отображение температуры и погодных условий непосредственно в верхней части экрана карты для удобства планирования логистики.

### 2.5. Управление заказами (Cart & Profile)
* **Корзина:** Гибкое управление списком покупок — добавление, удаление и пересчет стоимости в реальном времени.
* **Профиль:** Персональные настройки пользователя и история взаимодействия с приложением.

## 3. Технический стек

* **Язык программирования:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Навигация:** Navigation Compose (реализация Bottom Navigation Bar)
* **Работа с сетью:** Retrofit 2 + OkHttp
* **Сериализация данных:** Gson
* **Работа с картами:** osmdroid (OpenStreetMap SDK)
* **Архитектура:** Clean Architecture с элементами MVVM для разделения логики и представления.

## 4. Инструкция по сборке через CLI

Для компиляции проекта без использования тяжеловесных сред разработки (IDE) используется консольный интерфейс Gradle.

1.  **Подготовка:** Убедитесь, что установлена переменная окружения `JAVA_HOME` (JDK 17+).
2.  **Очистка проекта:**
    ```bash
    ./gradlew clean
    ```
3.  **Генерация APK:**
    ```bash
    ./gradlew assembleDebug
    ```
4.  **Результат:** Готовый исполняемый файл `app-debug.apk` будет сгенерирован в директории:
    `app/build/outputs/apk/debug/`

## 5. Требования к системе
* **ОС:** Android 7.0 (API level 24) и выше.
* **Разрешения:** Интернет (INTERNET), Доступ к местоположению (ACCESS_FINE_LOCATION), Сетевое состояние (ACCESS_NETWORK_STATE).
