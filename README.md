## 1. Обзор проекта
«Вкус» — это современное мобильное приложение для сегмента электронной коммерции, ориентированное на продажу полезных перекусов и продуктов здорового питания. Проект выполнен на базе фреймворка **Jetpack Compose**, что позволило реализовать отзывчивый интерфейс с динамическими анимациями и строгой архитектурной структурой.

Цель работы: разработка масштабируемого приложения с интеграцией внешних REST API и картографических сервисов для обеспечения пользователя актуальной контекстной информацией.

## 2. Функциональные возможности

### 2.1. Каталог продукции (Feed)
* **Динамическая лента:** Отображение товаров с разделением на смысловые блоки (популярное, новинки).
* **Интерактивные карточки:** Каждая позиция содержит визуальное представление, цену и краткую характеристику.
* **Экран деталей:** Полная информация о продукте, включая состав и КБЖУ.

### 2.2. Поиск и фильтрация (Search)
* Полнотекстовый поиск по ассортименту магазина.
* Группировка товаров по категориям для быстрой навигации.

### 2.3. Геосервисы и Картография (Map)
* **OpenStreetMap:** Модуль навигации для поиска точек выдачи заказов.
* **Смарт-маркеры:** Метки ключевых объектов (пункты самовывоза, офисы) с описанием.
* **LBS:** Реальное отображение текущей позиции пользователя на карте.

### 2.4. Метеорологический информер (Weather API)
* **Live-мониторинг:** Загрузка данных о погоде через OpenWeather API в реальном времени.
* **Контекстный UI:** Вывод температуры и условий (осадки, облачность) непосредственно на экран карты.

## 3. Руководство пользователя

### 3.1. Навигация
Управление приложением осуществляется через нижнюю панель навигации (Bottom Bar):
* **Домой:** Главная лента продуктов.
* **Поиск:** Поиск по каталогу.
* **Карта:** Интерактивная карта и погода.
* **Корзина:** Список покупок и управление количеством товаров.
* **Профиль:** Личный кабинет пользователя.

### 3.2. Работа с картой и точками выдачи
1. Перейдите в раздел **«Карта»** (иконка глобуса).
2. Для взаимодействия с картой используйте стандартные жесты: «щипок» для масштабирования и перемещение пальцем для навигации.
3. Нажмите на **маркер** (целевая точка), чтобы прочитать краткую информацию об объекте.
4. Ваше текущее местоположение отмечено **синим индикатором**. Если точка не отображается, проверьте, включен ли GPS на устройстве.

### 3.3. Получение данных о погоде
Данные о погоде обновляются автоматически при каждом открытии экрана карты. В верхней части экрана появится панель с текущим городом, температурой в градусах Цельсия и состоянием неба.

## 4. Технический стек

* **Язык:** Kotlin
* **UI:** Jetpack Compose (Material 3)
* **Сеть:** Retrofit 2 + OkHttp + Gson
* **Карты:** osmdroid (OpenStreetMap SDK)
* **Архитектура:** MVVM (Model-View-ViewModel)

---

## 5. Инструкция по сборке (CLI)

Сборка осуществляется через Gradle Wrapper. Этот метод позволяет скомпилировать проект без запуска тяжелых сред разработки.

1.  **Очистка проекта:**
    ```bash
    ./gradlew clean
    ```
2.  **Сборка APK:**
    ```bash
    ./gradlew assembleDebug
    ```
3.  **Локация файла:** После завершения (`BUILD SUCCESSFUL`) заберите файл по пути:
    `app/build/outputs/apk/debug/app-debug.apk`
